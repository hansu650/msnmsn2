# EviPatch Stage A

This directory contains project-owned code for evidence statistics, deterministic observation shifts, APN orchestration, statistical aggregation, and result packaging. The official APN source remains a pinned, ignored clone under `vendor/APN` and is modified only by the reproducible patch in `patches/`.

Environment and run commands are added as each component is verified. The full experiment order is smoke tests, PhysioNet Stage A, automatic gate, then conditional expansion.
